Installer macFUSE et SSHFS sur Mac

Cette page vise le chemin le plus court : installer macFUSE et SSHFS, choisir la bonne build, terminer l'autorisation initiale et tester le premier remote.

Télécharger la dernière versionOuvrir le dépannage

Étape 1 : installer les prérequis

Installez d'abord macFUSE puis sshfs. macFUSEGui dépend des deux.

brew install --cask macFUSE
brew install gromgit/fuse/sshfs-mac

Étape 2 : choisir la bonne build

Vérifiez l'architecture avec uname -m. arm64 pour Apple Silicon, x86_64 pour Intel.

Un mauvais téléchargement est l'une des erreurs d'installation les plus faciles à éviter.

uname -m

Étape 3 : terminer l'autorisation initiale

Ouvrez la build non signée une première fois depuis Finder.

Si macOS bloque encore, autorisez l'app dans Confidentialité et sécurité.

Étape 4 : ajouter le premier remote

  1. Renseigner host, utilisateur, mode d'authentification, chemin distant et point de montage local.
  2. Tester la connexion dans l'UI.
  3. Ouvrir le chemin monté dans Finder ou l'éditeur.

Étape 5 : si l'installation échoue

  • Revérifier macFUSE et sshfs.
  • Confirmer la bonne build.
  • Vérifier l'autorisation initiale.
  • Basculer vers le dépannage pour les problèmes d'auth ou de reprise.
