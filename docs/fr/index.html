Plus besoin de reconstruire sans cesse des commandes de montage fragiles. macFUSEGui fournit une macFUSE GUI dédiée à SSHFS sur macOS, avec des contrôles par remote, des identifiants dans Keychain, des diagnostics et une reprise après veille ou changement réseau.

SSHFS mounts with status, recovery, and macOS-native controls.

  • Diagnostics you can copy before escalating a mount failure.

Pourquoi les équipes choisissent macFUSEGui

L'application s'appuie sur macFUSE et sshfs pour que les dossiers distants se comportent comme des répertoires macOS normaux dans Finder et vos éditeurs.

Mise en route rapide

Enregistrez un remote une fois, testez la connexion dans l'UI et évitez les longues commandes SSHFS recopiées.

Reconnexion automatique

Les remotes voulus reviennent après veille, réveil, changement de Wi‑Fi et unmount externe.

Sécurité Keychain

Les mots de passe restent dans le trousseau macOS, tandis que la configuration non sensible reste en JSON.

Contrôle par remote

Connectez ou déconnectez un mount sans bloquer les autres remotes actifs.

Diagnostics

Copiez les vérifications d'environnement, les états et les logs récents avant de supposer.

Passage vers l'éditeur

Les dossiers montés s'ouvrent facilement dans Finder et dans le flux plugin des éditeurs.

Parcours d'installation et premier lancement

Installez d'abord macFUSE et sshfs, choisissez ensuite la build adaptée à votre CPU, puis terminez l'autorisation macOS au premier lancement.

Méthodes d'installation

Homebrew Cask

Idéal pour une installation et des mises à jour répétables.

brew tap ripplethor/macfusegui https://github.com/ripplethor/macfuseGUI && brew install --cask ripplethor/macfusegui/macfusegui

Installeur Terminal

Récupère l'installeur le plus récent et choisit l'artefact adapté à cette machine.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/ripplethor/macfuseGUI/main/scripts/install_release.sh)"

DMG manuel

Téléchargez le bon DMG pour votre Mac puis glissez l'app dans Applications.

Prérequis

macFUSEGui ne remplace pas macFUSE ni sshfs, il les orchestre. Pour un montage SSHFS fiable sur macOS, les deux doivent être installés.

Si votre environnement utilise déjà la formule Homebrew core, vous pouvez aussi essayer brew install sshfs après macFUSE.

brew install --cask macFUSE
brew install gromgit/fuse/sshfs-mac

Choisir le bon téléchargement

Apple Silicon utilise arm64, Intel utilise x86_64. Vérifiez l'architecture avant de télécharger.

Sur la page des releases, choisissez le DMG qui se termine par -macos-arm64.dmg ou -macos-x86_64.dmg.

uname -m

Fonctionnement

macFUSEGui est la couche de contrôle. macFUSE fournit le système de fichiers, sshfs le transport, et l'application orchestre la connexion, la reprise et les diagnostics.

  • Construit des commandes de montage sûres et gère connexion, déconnexion et recovery.
  • Stocke les mots de passe dans le trousseau macOS au lieu de l'historique shell.
  • Conserve la configuration non sensible dans ~/Library/Application Support/macfuseGui/remotes.json.
  • Expose des diagnostics copiables quand l'installation, l'authentification ou la reprise échouent.
