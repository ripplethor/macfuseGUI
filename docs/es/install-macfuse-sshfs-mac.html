Instalar macFUSE y SSHFS en Mac

Esta página prioriza el camino más corto: instalar macFUSE y SSHFS, elegir la build correcta, completar la aprobación inicial y probar el primer remote.

Descargar la última versiónAbrir solución de problemas

Paso 1: instalar los requisitos previos

Instala primero macFUSE y después sshfs. macFUSEGui depende de ambos.

brew install --cask macFUSE
brew install gromgit/fuse/sshfs-mac

Paso 2: elegir la build correcta

Usa uname -m para confirmar la arquitectura. arm64 es Apple Silicon; x86_64 es Intel.

Descargar la build equivocada sigue siendo un error común y evitable.

uname -m

Paso 3: completar la aprobación inicial

Abre una vez la build no firmada desde Finder con clic derecho.

Si macOS sigue bloqueando, autoriza la app en Privacidad y seguridad.

Paso 4: añadir el primer remote

  1. Rellenar host, usuario, modo de autenticación, ruta remota y punto de mount local.
  2. Probar la conexión en la UI.
  3. Abrir la ruta montada en Finder o en el editor.

Paso 5: si algo falla

  • Confirma de nuevo macFUSE y sshfs.
  • Verifica que la build descargada es la correcta.
  • Revisa la aprobación inicial.
  • Usa troubleshooting para auth, recovery y problemas de mount point.
