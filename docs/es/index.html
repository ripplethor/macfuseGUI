Por qué los equipos usan macFUSEGui

La app se sitúa sobre macFUSE y sshfs para que las carpetas remotas se comporten como directorios normales de macOS en Finder y editores.

Configuración rápida

Guarda remotes una sola vez, prueba la conexión en la UI y evita copiar comandos SSHFS largos.

Reconexión automática

Los remotes deseados vuelven tras reposo, activación, cambios de Wi‑Fi y desmontajes externos.

Seguridad con Keychain

Las contraseñas permanecen en el llavero de macOS y la configuración no sensible sigue en JSON.

Control por remote

Conecta o desconecta un mount sin bloquear los demás remotes activos.

Diagnósticos

Copia comprobaciones de entorno, estados y logs recientes antes de adivinar.

Paso a editores

Las carpetas montadas se pueden abrir directamente en Finder y en el flujo de plugins de editor.

Métodos de instalación

Homebrew Cask

Ideal para instalaciones y actualizaciones repetibles.

brew tap ripplethor/macfusegui https://github.com/ripplethor/macfuseGUI && brew install --cask ripplethor/macfusegui/macfusegui

Instalador por Terminal

Obtiene el instalador más reciente y elige el artefacto adecuado para esta máquina.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/ripplethor/macfuseGUI/main/scripts/install_release.sh)"

DMG manual

Descarga el DMG correcto para tu Mac y arrastra la app a Applications.

Requisitos previos

macFUSEGui no sustituye a macFUSE ni a sshfs; los orquesta. Para mounts SSHFS fiables en macOS necesitas ambos instalados.

Si tu entorno ya usa la fórmula de Homebrew core, también puedes probar brew install sshfs después de macFUSE.

brew install --cask macFUSE
brew install gromgit/fuse/sshfs-mac

Elige la descarga correcta

Apple Silicon usa arm64; Intel usa x86_64. Verifica la arquitectura antes de descargar.

En la página de releases, elige el DMG que termina en -macos-arm64.dmg o -macos-x86_64.dmg.

uname -m

Cómo funciona

macFUSEGui es la capa de control. macFUSE aporta el sistema de archivos, sshfs el transporte y la app gestiona conexión, recuperación y diagnósticos.

  • Construye comandos de mount seguros y gestiona conectar, desconectar y recovery.
  • Guarda contraseñas en el llavero de macOS en vez del historial del shell.
  • Mantiene la configuración no sensible en ~/Library/Application Support/macfuseGui/remotes.json.
  • Entrega diagnósticos copiables cuando fallan instalación, autenticación o recovery.
