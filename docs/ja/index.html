macFUSEGui が選ばれる理由

macFUSEGui は macFUSE と sshfs の上に位置し、リモートフォルダを Finder やエディタでローカルに近い感覚で扱えるようにします。

すばやいセットアップ

インストール方法

Homebrew Cask

再現性のあるインストールと更新経路が欲しい場合に最適です。

brew tap ripplethor/macfusegui https://github.com/ripplethor/macfuseGUI && brew install --cask ripplethor/macfusegui/macfusegui

ターミナルインストーラ

最新リリースのインストーラスクリプトを取得し、この Mac に合う成果物を選ばせます。

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/ripplethor/macfuseGUI/main/scripts/install_release.sh)"

手動 DMG

自分の Mac に合う DMG をダウンロードして Applications にドラッグします。

最新リリースを開く

前提条件

macFUSEGui は macFUSE や sshfs を置き換えるものではなく、それらを管理するレイヤーです。安定した SSHFS マウントには両方の導入が必要です。

すでに Homebrew core の formula を使っている環境なら、macFUSE の後に brew install sshfs を試すこともできます。

brew install --cask macFUSE
brew install gromgit/fuse/sshfs-mac

正しいダウンロードを選ぶ

Apple Silicon は arm64、Intel は x86_64 です。まずアーキテクチャを確認してください。

リリースページでは、-macos-arm64.dmg または -macos-x86_64.dmg で終わる DMG を選びます。

uname -m

仕組み

macFUSEGui は制御レイヤーです。macFUSE がファイルシステム、sshfs が転送を担当し、その上でアプリが接続、回復、診断を扱います。

  • 安全なマウントコマンドを組み立て、接続・切断・回復を管理します。
  • パスワードをシェル履歴ではなく macOS Keychain に保存します。
  • 非機密の設定は ~/Library/Application Support/macfuseGui/remotes.json に保存されます。
  • インストール、認証、回復の失敗時にコピー可能な診断を出力します。
