Mac で macFUSE と SSHFS をインストール

このページは最短導入向けです。macFUSE と SSHFS を入れ、正しいビルドを選び、初回承認を済ませて、最初のリモートをテストします。

最新リリースをダウンロードトラブルシューティングを開く

ステップ 1: 前提条件をインストール

最初に macFUSE、次に sshfs をインストールします。macFUSEGui は両方に依存しています。

brew install --cask macFUSE
brew install gromgit/fuse/sshfs-mac

ステップ 2: 正しいビルドを選ぶ

uname -m で CPU アーキテクチャを確認します。arm64 は Apple Silicon、x86_64 は Intel です。

別のマシンからリンクを受け取った場合は、間違ったビルドを落としやすいので注意してください。

uname -m

ステップ 3: 初回起動の承認を完了

現在の公開ビルドは未署名です。Finder で右クリックして「開く」を選びます。

まだブロックされる場合は「システム設定 > プライバシーとセキュリティ」で許可してください。

ステップ 4: 最初のリモートを追加

  1. ホスト、ユーザー名、認証方式、リモートパス、ローカルマウントポイントを入力する。
  2. Finder に頼る前に UI から接続テストを行う。
  3. 接続後に Finder またはエディタでマウントパスを開く。

ステップ 5: うまくいかないとき

  • macFUSE と sshfs が両方インストールされているか再確認する。
  • 自分の Mac に合うビルドを選んだか確認する。
  • 初回承認が完了しているかを確かめる。
  • 認証や再接続の問題はトラブルシューティングへ進む。
